# Criska-Timeline
The whole life of these two men

欢迎来到本项目。本仓库旨在客观、中立地梳理当事人 A 与当事人 B 之间的事件演变脉络。

### 🌐 正式访问入口
👉 **[点击此处立刻进入交互式时间线网页](https://luna-chestnuty.github.io/Criska-Timeline/)**
*(建议在电脑端或手机端直接打开，支持横向时间轴拖拽、分类筛选及点击展开详细证据链)*

---

### 📂 项目说明
* **数据来源**：所有时间节点均附带原始超链接支持，资料来源于双方社交媒体公开内容及媒体报道。
* **排版功能**：
  * **电脑端**：支持横向时间轴左右滑动，上半部分可直观对比左右双轨（A与B）的行动轨迹。
  * **手机端**：自动优化为垂直单轨流，方便使用大拇指顺畅下滑阅读。
  * **交互操作**：点击时间轴上的标签可切换事件，点击正上方菜单可按分类筛选特定事件分类。
